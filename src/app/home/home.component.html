TDD (Technical Design Document)

Bookmark using Angular JS

The application itself is a single page and it's main components is named registration (for entering and displaying a bookmark list).

Constraints

The form makes use of Template-Driven validation in opposite to Reactive forms. I believe reactive form validation would be more efficient with URL but I used a let checkFlag: boolean = false; for checking bookmarks bookmarkName (see registration component.ts) however the validation is not fullfilling http as I used pattern="https://.*"
